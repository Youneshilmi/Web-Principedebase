Site E-commerce complet

Technologies : HTML, CSS, JavaScript, Node.js

Développement d'une plateforme e-commerce de vente de services complète avec système de panier, gestion des utilisateurs et interface d'administration. Le site permet aux visiteurs de parcourir un catalogue de produits, d'ajouter des articles au panier et de simuler un processus de commande.

Fonctionnalités principales :

  • Catalogue de produits avec filtres et recherche
  • Système de panier dynamique
  • Gestion des utilisateurs et authentification JWT
  • Interface responsive pour tous les appareils
  • Backend Node.js avec API REST

Gestionnaire de tâches intelligent

Technologies : Python, Tkinter

Application desktop développée en Python permettant de gérer efficacement ses tâches quotidiennes. L'application offre une interface intuitive pour créer, organiser et suivre l'avancement de ses projets personnels ou professionnels.

Fonctionnalités principales :

  • Création et organisation de tâches par catégories
  • Système de priorités et de dates limites
  • Notifications pour les tâches importantes
  • Export des données en format CSV/XLSX
  • Interface graphique ergonomique

ERP local

Technologies : Python, Django

Développement d'un ERP local performant pour la gestion des ressources et des opérations. Ce système intègre plusieurs modules pour la gestion des stocks, des ventes, des achats et des ressources humaines.

Fonctionnalités principales :

  • CRUD complet pour articles et commentaires
  • Authentification JWT sécurisée
  • Gestion des rôles utilisateurs
  • Upload d'images pour les articles
  • Documentation automatique avec Swagger

Visualiseur d'algorithmes de tri

Technologies : C++, SDL2

Application en C++ permettant de visualiser en temps réel le fonctionnement de différents algorithmes de tri. Projet éducatif qui aide à comprendre la complexité et l'efficacité des algorithmes classiques de tri.

Fonctionnalités principales :

  • Visualisation de 6 algorithmes de tri différents
  • Animation en temps réel du processus de tri
  • Comparaison des performances
  • Génération de données aléatoires
  • Interface graphique avec SDL2
